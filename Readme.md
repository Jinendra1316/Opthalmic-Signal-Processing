**Pattern Electroretinogram (PERG) signals** are beneficial for diagnosing retinal and optic nerve disorders, as they provide an objective evaluation of retinal ganglion cell and macular function. This study utilises the recently released **PERG-IOBA dataset**, comprising 1,354 transient PERG signals collected from 304 subjects at the University of Valladolid, Spain, between 2003 and 2022. The project aims to **develop machine learning models capable of predicting visual acuity (logMAR) and classifying ocular health status (normal versus pathological) directly from raw PERG waveforms.**

The proposed framework encompasses signal preprocessing, feature extraction from distinctive waveform components (N35, P50, and N95), and the application of both classical machine learning and deep learning techniques, including convolutional and recurrent neural network architectures. Model performance is evaluated using standard regression and classification metrics such as Root Mean Squared Error (RMSE), accuracy, and the Area Under the Receiver Operating Characteristic Curve (ROC-AUC).

By establishing a relationship between electrophysiological retinal signals and functional visual outcomes, this work aims to improve the early detection of vision loss and automate the interpretation of PERG tests. Furthermore, the study demonstrates how open biomedical datasets can facilitate advancements in ophthalmic signal analysis through data-driven modelling approaches.

**Index Terms:** Pattern Electroretinogram (PERG), PERG-IOBA Dataset, Ophthalmic Signal Processing, Machine Learning, Deep Learning, Biomedical Signal Classification, Retinal Diagnostics, Visual Acuity Prediction.
